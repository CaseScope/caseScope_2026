** SITE LAYOUT **
- /app: all site application files.
- /app/routes: all routes; these should be auto adding in nature so new routes get pulled into the application automatically.
- /css: all CSS files, the site should centralize as much as possible into this CCS for uniformity.
- /templates: every page should have a template stored here.
- /ai_prompts: all AI prompts created will be in template form and placed here so the end user can adjust if needed.
- /bin: site executables like chainsaw and so on go here
