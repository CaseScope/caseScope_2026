Investigative Summary
---------------------
[WARNING]
Please review this incident report to understand what was identified before remediating. There may be unknown malicious processes, files, or other changes made to the host that remain undetected. Restoring from a known good backup or clean OS install is the only way to ensure a complete host-level remediation. 

Evidence suggests that on 2025-03-05 at 13:17 UTC, the user 'Kwork' executed a malicious powershell script that reached out to the IP '64[.]94[.]84[.]217' to download additional malicious payloads. 

This led to a malicious install of node.exe in the directory 'C:\Users\KWork\AppData\Roaming\node-v22.11.0-win-x64\', this has been added to assisted remediations for removal. 

Due to the severity of this incident, this host has been isolated to prevent further ingress. 

Huntress detected the following:
- Powershell Downloader : Huntress analysts found a malicious powershell command used to download a payload from a remote server. This is likely being used to deploy malware onto the victim machine. 
- Powershell Payload : This is a Powershell command which extracts a malicious file stored in the registry and executes it. The malicious file is used to spread Ursnif, a banking trojan, which is capable of web browser manipulation, web injections, man-in-the-browser functionality, form grabbing, screen capture, and hidden VNC and SOCKS proxy attacks.

Malicious Downloader / Malicious Download: Malicious activity has resulted in the download and execution of an additional malicious payload(s).
