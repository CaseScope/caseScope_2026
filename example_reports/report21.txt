Investigative Summary
---------------------
On 2025-09-25 at 04:07:28 UTC, Huntress detected a malicious ScreenConnect wscript executing from the temporary directory on the device "Edward-W10". This ScreenConnect instance was confirmed to be malicious and was observed reaching out to the domain ind.erdemliecotourism[.]com. At the same time, Huntress noted the concurrent installation of a malicious PDQ RMM tool that was installed on 2025-08-25 at 19:10:17 UTC. 

Further analysis, we noticed multiple RMM tools added to the device. This includes the following:
- JWrapper-Remote Access.
- SimpleHelp.
- CentraStage.
- RemotePC.

As part of the investigation into the original activity, Huntress conducted static analysis on the file "C:\Users\Edward\Documents\ConnectWiseControl\Temp\Steepwe.js". This revealed an obfuscated script, which we successfully decoded. The script was found to point to another file on the endpoint located at:
"C:\Users\Edward\AppData\Roaming\Wriedrepr"

This second file was also obfuscated. After Huntress decoded it, we determined that it made a malicious GET request to the following domain:
http[://]drive[.]sourcekity-cloud.io/c3/Abo.ktf

We recommend you wipe this device and restore it from a known good backup. 

Threat Descriptions:
-------------------
Rogue ScreenConnect: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious ScreenConnect (ConnectWise Control) installers that give the threat actor remote access to the host. 

Malicious Downloader / Malicious Download: Malicious activity has resulted in the download and execution of an additional malicious payload(s). 

Malicious Script: A malicious script is a type of code that can be used to harm a computer without the user's knowledge. Malicious scripts are a type of malicious code, which is any computer programming or web scripting that is designed to cause damage or create security holes.

Suspicious App Usage: Huntress has detected a suspicious enterprise application added to this tenant.

Suspicious RMM: A suspicious remote monitoring and management (RMM) application was detected on this endpoint. These applications may be benign within the environment, or levied by adversaries as a method of persistence. Occasionally software vendors will additionally distribute these RMM's as a portion of their support process. 

Malicious Remote Management Tool: A legitimate remote management tool has been misused by a threat actor to gain unauthorized access and control of the reported endpoint. Threat actors do this to hide their presence, maintain persistence and further exploit the network.

Rogue Simplehelp: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious Simplehelp installers that give the threat actor remote access to the host.
