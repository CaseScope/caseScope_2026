Investigative Summary
---------------------
At 2025-08-19 18:05:08 UTC, Huntress observed user "User1" executing a malicious process ScreenConnect.ClientService.exe. Huntress observed this user downloaded remittss.exe on the endpoint ATN85406. This interaction resulted in multiple remote management tool installations that connect to the following domains:
suyafisi[.]top
https[:]//funksstarter[.]top

It is recommended to audit all of the remote management tools installed on the endpoint in addition to following the remediation section.

Threat Descriptions:
-------------------
Rogue ScreenConnect: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious ScreenConnect (ConnectWise Control) installers that give the threat actor remote access to the host. 

Rogue Simplehelp: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious Simplehelp installers that give the threat actor remote access to the host.

Persistence : Persistence consists of techniques that adversaries use to keep access to systems across restarts, changed credentials, and other interruptions that could cut off their access. Techniques used for persistence include any access, action, or configuration changes that let them maintain their foothold on systems, such as replacing or hijacking legitimate code or adding startup code.
