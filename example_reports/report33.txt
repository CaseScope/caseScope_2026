Investigative Summary
---------------------
At 2025-08-15 16:12:16 UTC, Huntress observed a ScreenConnect instance to a known malicious domain asfsa1236ygfdg[.]anondns[.]net. This ScreenConnect instance was installed by a PowerShell script launched from another RMM, Datto CentraStage AEMAgent.
This agent created multiple firewall rules to allow several Remote Management tools outbound, including UltraVNC Viewer.

Threat Descriptions:
-------------------
Rogue ScreenConnect: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious ScreenConnect (ConnectWise Control) installers that give the threat actor remote access to the host.

-------------
