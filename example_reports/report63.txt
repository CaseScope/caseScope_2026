[WARNING]
Please review this incident report to understand what was identified before remediating. There may be unknown malicious processes, files, or other changes made to the host that remain undetected. Restoring from a known good backup or clean OS install is the only way to ensure a complete host-level remediation. 

Huntress detected the following:
- Rogue ScreenConnect Install : Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious ScreenConnect (ConnectWise Control) installers that give the threat actor remote access to the host.

On 2025-02-24 15:29:49 UTC, user "kmeccariello" was observed interacting with malicious file "C:\Users\kmeccariello\Downloads\DocInvoiice2025feb24.exe". This file, a renamed ScreenConnect remote desktop agent, is configured to communicate with adversarial services and facilitate remote access to the environment. 

This ScreenConnect agent is configured to communicate with the following domain which does not appear to be the standard management site used by this organization:
- ushelp[.]top

**NOTE: Additional reports may follow to address persistence for this ScreenConnect agent. Remediation instructions present are sufficient to continue normal business use once completed. **

Remediation Instructions
------------------------
- Remove the "C:\Users\kmeccariello\Downloads\DocInvoiice2025feb24.exe" file.
- Remove the "C:\Program Files (x86)\ScreenConnect Client (de5851ad6e374ce3)\" directory. 
- Reboot the host to terminate outstanding malicious processes.
