Investigative Summary
---------------------
Huntress has investigated alerts for the host "ATN82204"

Evidence suggests that, while browsing the website "hxxps[://]nexusvalve[.]com/contact-us" with Google Chrome, the user "JRowe" (S-1-5-21-1849904498-4039813688-2514114501-5629) clicked on a malicious advert and downloaded a malicious Javascript file "Update 7985.js" and on 2025-02-20 at 13:33:15 UTC they executed the malicious file, infecting the host with the SocGholish malware.

To prevent further malicious activity, Huntress has isolated the host.

Huntress recommends restoring from a known good backup prior to the time of infection .

Huntress also recommends enrolling the user "JRowe" onto security awareness training to prevent similar incidents from occurring in the future. 


[WARNING]
Please review this incident report to understand what was identified before remediating. There may be unknown malicious processes, files, or other changes made to the host that remain undetected. Restoring from a known good backup or clean OS install is the only way to ensure a complete host-level remediation. While the Assisted Remediation service is an option, it will only remove specific items documented in this report.

Huntress detected the following:
- SocGholish / FakeUpdates : SocGholish, also known as FakeUpdates, is used by threat actors to deliver malicious payloads by tricking users to run malware that is disguised as a legitimate software update.
- Hidden NetSupport Install : This remote desktop application from https://www.netsupportsoftware.com/ is running from a non-standard location. The application is normally located in "c:\program files" and it has been renamed to blend in. Given the odd location of the software and since there are other remote access tools installed on the host this application seems out of place.
