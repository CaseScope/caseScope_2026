Investigative Summary
---------------------
On 2025-08-07 at 14:05:01 UTC, a malicious Windows startup with the name "PP1" was detected on the endpoint "Timsassistant". This startup will execute a malicious NetSupport instance (C:\Users\JacobParks\AppData\Roaming\Dire\client32.exe) configured to communicate with the adversarial IP address "94.158.245[.]140" as a persistence mechanism. Available telemetry indicates that this startup was present prior to the installation of the Huntress agent, suggesting that this endpoint may have been previously compromised. No additional evidence of malice was observed within available data at the time of investigation.

Threat Descriptions:
-------------------
Hidden NetSupport: NetSupport is a legitimate remote management tool that is misused by threat actors to gain unauthorized access and control of the reported endpoint. Threat actors do this to hide their presence, maintain persistence, and further exploit the network.

Persistence : Persistence consists of techniques that adversaries use to keep access to systems across restarts, changed credentials, and other interruptions that could cut off their access. Techniques used for persistence include any access, action, or configuration changes that let them maintain their foothold on systems, such as replacing or hijacking legitimate code or adding startup code.

Prior Compromise: The security incident on this host occurred prior to the Huntress agent installation. We will attempt to address any observed malicious behavior, but there may be unreported changes to this host that will not be detected due to limited telemetry.

Malware Artifact: A Malware Artifact is a piece of evidence left behind by malware on a computer system. It could be a file, registry entry, or network connection that indicates the presence of malicious activity.
