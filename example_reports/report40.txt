---------------------
At 2025-07-23 15:54:13 UTC, Huntress observed user "mdillon" executing a malicious process igfxSDK.exe.

Evidence suggests the user "mdillon" interacted with the malicious file "igfxSDK.exe" which is part of "Lunar" malware, which is known to be malicious. We highly suggest resetting all user credentials and restoring from a known good backup. 

Threat Descriptions:
-------------------
Backdoor IFEO: A backdoor using IFEO (Image File Execution Options) is a technique that allows threat actors to maintain access to a compromised system. IFEO is a Windows feature that lets developers specify a debugger to use when a program crashes, however, threat actors can misuse this feature to launch their own malicious programs instead of the intended debugger.

The Huntress SOC identified the Malware Families as QakBot and Dridex.
