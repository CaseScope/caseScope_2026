Investigative Summary
---------------------
Huntress detected the following: 

Evidence suggests that on 2025-07-17 the host "DRA" joined Huntress with artifacts from a malware infection which predates the Huntress agent installation. 

Two autorun values titled "Bqrgro" and "Dhputgczz" were identified, which were instructed to run the following executables: 

 - C:\Users\cjohnson\AppData\Roaming\Microsoft\PowerPoint\WA5A\tabcal.exe
 - C:\Users\cjohnson\AppData\Roaming\Microsoft\Protect\Dac\Dxpserver.exe

Please follow the assisted remediations to remove these artefacts from the host. 

The Huntress SOC identified the Malware Family as Dridex.
