Investigative Summary
---------------------
At 2025-09-05 05:51:18 UTC, the Huntress agent on host "Receiving" received commands via a Type 3 / network login from un-managed source host "192.168.1.150", under the user context "JAMESMFG\tabadmin", to enable Remote Desktop access to this host. The actual time of these commands were from 05:43 UTC.

Once enabled, the source user then authenticated via a Type 10 / Remote Interactive (RDP) logon to this host at 05:49 UTC.

We then reviewed the surrounding events, and those in near-real time only to observe attempts from this logon to exclude the System Drive C:\ from Windows Defender scans, a common method of attack from actors to avoid detection. As such, at this point, we network isolated the host at 06:04 UTC.

Please validate whether the activity related to `tabadmin` was sanctioned and follow proper containment and recovery protocols to identify and isolate source host "192.168.1.150", as well as rotate credentials for the user in question.

Threat Descriptions:
-------------------
Lateral Movement: Lateral movement involves the threat actor pivoting across machines. 

Disabling or Modifying Security Tools on Endpoint: The threat actor has disabled or modified security tooling in an attempt to bypass controls and further compromise the network.

Remediations:
-------------
Manual Remediations provided by the Huntress SOC are highly recommended remediation actions to be conducted by your team before resolving the incident in the Huntress Platform:
- Consider enacting incident response procedures to assist in fully scoping this security incident.
- Disable and/or rotate credentials for any affected user accounts.
- Disable the firewall rule enabling RDP access, if this rule (group="remote desktop") is not explicitly required on this host.
- Audit network perimeter device logs such as firewalls, remote desktop services, etc. for anomalous/unauthorized access.
- Audit local endpoint logging on affected hosts for anomalous access times or unanticipated activity.
- Deploy the Huntress agent to all endpoints in the organization to increase visibility across the network for the Huntress SOC.
- Ensure endpoint anti-virus solutions are deployed and are functioning properly. Audit these services for alerts or warnings on affected hosts.

All remediations provided can be found in the Huntress Platform: Incident Report: https://tabinc.huntress.io/org/150377/infection_reports/1682134#remediations-tab
