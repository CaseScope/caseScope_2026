Investigative Summary
---------------------
At 2025-09-11 14:56:11 UTC, Huntress observed user "usedsales" [S-1-5-21-3694373414-3881711598-125369868-1107] executing a malicious process "support.ClientSetup.msi." Evidence suggests that this is a rogue ScreenConnect installation configured to communicate with "onyxcrytolive[.]top," with an anomalous connection observed going outbound to "50[.]225[.]154[.]136" after execution. Due to the nature of this activity, we have elected to isolate the endpoint. We recommend removing all associated files and directories, reimaging the host, and enrolling the end user in security awareness training.

Threat Descriptions:
-------------------
Rogue ScreenConnect: Huntress has been tracking a number of malicious threat actors convincing users via email into running malicious ScreenConnect (ConnectWise Control) installers that give the threat actor remote access to the host.
