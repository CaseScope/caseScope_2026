💡 How Exclusions Work

When AI Triage Search hunts for suspicious activity, it can match legitimate tools used by your IT team. Configure exclusions here to prevent false positives:

  • RMM Tools: Events spawned by these processes (e.g., LabTech running whoami) are fully excluded
  • Remote Tools: Sessions with known-good IDs are excluded; unknown IDs are still flagged
  • EDR/Security Tools: Routine health checks excluded, but isolation/response actions KEPT
  • Known-Good IPs: Events from internal networks or analyst IPs are excluded

Huntress, SentinelOne, CrowdStrike, etc. - Exclude routine checks, KEEP response actions
