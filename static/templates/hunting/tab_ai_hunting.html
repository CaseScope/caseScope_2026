🔗 Related Event Hunting

Find events contextually related to SIGMA hits, IOC matches, or analyst-tagged events.

❓ What does this do?

Related Event Hunting uses "anchor" events (high-severity SIGMA hits) to find surrounding context.

  • Anchor Events: High/Critical SIGMA detections serve as starting points
  • Time Window: Searches for related events within the specified time range around each anchor
  • Related Events: Finds logins, process executions, service installs, and other interesting events near the anchors

💡 This helps analysts understand what happened before and after a detection - crucial for building attack timelines.

📅 Timeline Generation

Generate MITRE ATT&CK mapped incident timeline using AI analysis.

❓ What does this do?

Timeline Generation creates an AI-powered incident narrative from your case events.

  • Phase Clustering: Groups events into attack phases based on time gaps (2+ hours between phases)
  • MITRE Mapping: Maps each phase to ATT&CK tactics and techniques
  • AI Narratives: Uses Ollama LLM to generate human-readable descriptions of attacker objectives
  • Executive Summary: Produces a high-level overview suitable for management reporting

💡 Best results when run after Pattern Discovery - pattern matches are included in the timeline.
