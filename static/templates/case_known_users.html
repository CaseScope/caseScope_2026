❓ PowerShell Scripts for User Export

Export Domain Users to CSV

Run this PowerShell script on a domain-joined system to export Active Directory users:

# Export Domain Users to CSV
# Requires: RSAT AD PowerShell module
# Run as Administrator on a domain-joined system

Import-Module ActiveDirectory

$OutputPath = "$env:USERPROFILE\Desktop\domain_users.csv"

Get-ADUser -Filter * -Properties SamAccountName, SID, EmailAddress, Enabled |
    Select-Object @{N='username';E={$_.SamAccountName}},
                  @{N='sid';E={$_.SID.Value}},
                  @{N='email';E={$_.EmailAddress}},
                  @{N='compromised';E={'false'}},
                  @{N='notes';E={''}} |
    Export-Csv -Path $OutputPath -NoTypeInformation

Write-Host "Exported to: $OutputPath" -ForegroundColor Green

Export Local Users to CSV

Run this PowerShell script to export local users from a Windows system:

# Export Local Users to CSV
# Run as Administrator

$OutputPath = "$env:USERPROFILE\Desktop\local_users.csv"

Get-LocalUser |
    Select-Object @{N='username';E={$_.Name}},
                  @{N='sid';E={$_.SID.Value}},
                  @{N='email';E={''}},
                  @{N='compromised';E={'false'}},
                  @{N='notes';E={$_.Description}} |
    Export-Csv -Path $OutputPath -NoTypeInformation

Write-Host "Exported to: $OutputPath" -ForegroundColor Green
