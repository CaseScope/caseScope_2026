❓ PowerShell Scripts for System Export

Export Domain Computers to CSV

Run this PowerShell script on a domain-joined system to export Active Directory computers:

# Export Domain Computers to CSV
# Requires: RSAT AD PowerShell module
# Run as Administrator on a domain-joined system

Import-Module ActiveDirectory

$OutputPath = "$env:USERPROFILE\Desktop\domain_systems.csv"

Get-ADComputer -Filter * -Properties Name, OperatingSystem, OperatingSystemVersion, IPv4Address |
    Select-Object @{N='hostname';E={$_.Name}},
                  @{N='system_type';E={if($_.OperatingSystem -match 'Server'){'Server'}else{'Workstation'}}},
                  @{N='os_type';E={'Windows'}},
                  @{N='os_version';E={$_.OperatingSystem}},
                  @{N='compromised';E={'false'}},
                  @{N='notes';E={''}},
                  @{N='ip_addresses';E={$_.IPv4Address}} |
    Export-Csv -Path $OutputPath -NoTypeInformation

Write-Host "Exported to: $OutputPath" -ForegroundColor Green

Export Local System Info to CSV

Run this PowerShell script to export the local system's information:

# Export Local System Info to CSV
# Run as Administrator

$OutputPath = "$env:USERPROFILE\Desktop\local_system.csv"

$os = Get-CimInstance Win32_OperatingSystem
$cs = Get-CimInstance Win32_ComputerSystem
$ips = (Get-NetIPAddress -AddressFamily IPv4 | 
        Where-Object {$_.IPAddress -notmatch '^(127\.|169\.254\.)'}).IPAddress -join ';'

[PSCustomObject]@{
    hostname = $env:COMPUTERNAME
    system_type = if($os.ProductType -eq 1){'Workstation'}else{'Server'}
    os_type = 'Windows'
    os_version = $os.Caption
    compromised = 'false'
    notes = ''
    ip_addresses = $ips
} | Export-Csv -Path $OutputPath -NoTypeInformation

Write-Host "Exported to: $OutputPath" -ForegroundColor Green
